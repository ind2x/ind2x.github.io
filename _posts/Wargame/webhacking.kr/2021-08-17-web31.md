## web31
```
$port = rand(10000,10100);
$socket = fsockopen($_GET['server'],$port,$errno,$errstr,3) or die("error : {$errstr}");
```

## Solution
```
nc로 모든 포트를 열어서 listening 했는데 여기 환경 상 풀 수 없어서 나중에 다시 풀기

우선 문제를 보면 내 IP주소로 되어있으며 포트는 10000~10100 사이로 된 서버로 들어오고자 하지만 들어오지 못함.
그래서 클라이언트 측에서 해당 포트번호들을 listening 상태로 열어둬야 함. 
즉, port forwarding을 해줘야 함.  열어준 뒤 nc로 내용을 확인해주면 플래그가 나옴.
```
