## scandir()

```php
scandir ( string $directory ) : array

## ex) websec level 8
print_r(scandir('/'));

Array
(
    [0] => .
    [1] => ..
    [2] => flag.txt
    [3] => index.php
    [4] => php-fpm.sock
    [5] => source.php
    [6] => uploads
)

지정된 디렉토리의 파일 및 디렉토리의 배열을 반환.
```


## file_get_contents()

```php
file_get_contents( string $filename ) : string

# ex) 
echo file_get_contents('flag.php');
print_r(file_get_contents('http://websec/level28/index.php'));
```

## printr_r()
```php
print_r ( mixed $expression [, bool $return = FALSE ] ) : mixed

#ex) 
$a = array ('a' => '1', 'b' => '2', 'c' => array ('3', '4', 'hi'));
print_r ($a);
# $results = print_r($b, true); 
Array
(
    [a] => 1
    [b] => 2
    [c] => Array
        (
            [0] => 3
            [1] => 4
            [2] => hi
        )
)

변수에 관한 정보를 사람이 읽기 편하게 출력, 
두번째 인자로 true를 써주면 변수에 결과를 저장할 수 있음.
```


## var_dump()
```php
var_dump ( mixed $expression [, mixed $... ] ) : void
# print_r()의 결과값에서 데이터형을 추가로 더 보여준다고 보면 됨.

$a = array ('a' => '1', 'b' => '2', 'c' => array ('3', '4', 'hi'));
Array
(
    [a] => 
    int(1)
    [b] => 
    int(2)
    [c] => 
    array(3)
        (
            [0] => 
            int(3)
            [1] => 
            int(4)
            [2] => 
            string(2) "hi"
        )
)

변수의 대한 정보를 덤프함. 
리턴값은 없음.
```


## var_export()
```php
var_export ( mixed $expression [, bool $return = FALSE ] ) : mixed

#ex)
$a = array ('a' => '1', 'b' => '2', 'c' => array ('3', '4', 'hi'));
array (
  0 => 1,
  1 => 2,
  2 => 
  array (
    0 => 3,
    1 => 4,
    2 => 'hi',
  ),
)

변수를 처리가능한 문자열 표현으로 출력하거나 반환함. 
var_dump()와의 차이는 var_export()는 유효한 PHP 코드를 반환한다는 점. 
$return을 TRUE로 하면 변수에 저장가능.
```
